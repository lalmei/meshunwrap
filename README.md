# meshunwrap

[![documentation](https://img.shields.io/badge/docs-mkdocs-708FCC.svg?style=flat)](https://lalmei.gi.io/meshunwrap/)
[![pypi version](https://img.shields.io/pypi/v/meshunwrap.svg)](https://pypi.org/project/meshunwrap/)
UV unwrapping and figure reproduction for tube-like meshes

## Installation

```bash
pip install meshunwrap
```

With [`uv`](https://docs.astral.sh/uv/):

```bash
uv tool install meshunwrap
```
